# ‘数组操作，索引与寻址
A(r,c)用定义r和c索引向量来寻址 A 的子数组
A(r,:)用r向量定义的行与其他所有的列得到A的子数组
A(:,C)用c向量定义的列与其他所有的行得到A的子数组
A(:) 得到A中所有元素生成的子数组，按列优先排列
A(X) 寻址数组A中第x个元素
